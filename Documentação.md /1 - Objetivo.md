# Objetivo
O objetivo principal deste projeto é desenvolver um jogo interativo de "Jogo da Discórdia" implementado em Python, agora com uma interface gráfica de usuário (GUI) utilizando customtkinter e uma arquitetura modular bem definida. O jogo permite que múltiplos jogadores participem, votando uns nos outros com base em perguntas polêmicas e fornecendo justificativas para seus votos. O sistema é responsável por gerenciar as rodadas, registrar os votos e as justificativas, e apresentar um ranking final dos jogadores com base na pontuação acumulada.

O foco é proporcionar uma experiência de jogo visualmente mais agradável e intuitiva através de uma GUI, incentivando a interação e a discussão entre os participantes, enquanto mantém uma estrutura de código organizada e de fácil manutenção, separando as responsabilidades entre Modelo, Controle e Visão (Model-View-Controller - MVC).
